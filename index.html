A customized audio player that works with either SoundCloud URLs or regular MP3 files. When using SoundCloud URLs there is the option to output the SoundCloud waveform thanks to waveform.js

Features also include a lot of customization of visual output and the capability to use as a persistent audio player on a site.

Developed for BigSushi.fm where you can see examples of it in production.

Special Thanks:
GitHub User: kilokeith. In which the majority of this plugin is based on his soundcloud-soundmanager-player

Scripts

  • jQuery (>= 1.7)
  • SoundCloud JS SDK
  • SoundManager2
  • waveform.js (optional)
  • Sushi Player (what you're here for)

Configure

In sushi-player.js go to the bottom of the file and insert your SoundCloud API Consumer Key. If you need a consumer key go to SoundCloud and Register a New Application .

Initialize

$('a.sushi-player').sushiplayer();

Defaults

$.fn.sushiplayer.defaults = {
  consumer_key: "", /* your soundcloud consumer key */
  permalink: null,
  scrubtype: 'bar',
  artworkurl: null,
  artwork: '<div class="sp-artwork"></div>',
  title: '<div class="sp-title"></div>',
  controls: '<div class="sp-controls">
              <a href="#play" class="sp-play"><i class="fa fa-play"></i></a> 
              <a href="#pause" class="sp-pause"><i class="fa fa-pause"></i></a>
            </div>',
  scrubber: '<div class="sp-scrubber">
              <div class="buffer"></div>
              <div class="played">
                <span class="time currenttime"></span>
              </div>
              <span class="time totaltime"></span>
            </div>'
}

Configurable options

You may change the defaults in the plugin or override individual options inline by using data attributes. Simply append the option name to data-, as in data-permalink.
Name Type Default Description
permalink URL null
scrubtype string bar
artworkurl URL null
