A customized audio player that works with either SoundCloud URLs or regular MP3 files. When using SoundCloud URLs there is the option to output the SoundCloud waveform thanks to waveform.js

Features also include a lot of customization of visual output and the capability to use as a persistent audio player on a site.

Developed for BigSushi.fm where you can see examples of it in production.

Special Thanks:
GitHub User: kilokeith. In which the majority of this plugin is based on his soundcloud-soundmanager-player

Scripts

  • jQuery (>= 1.7)
  • SoundCloud JS SDK
  • SoundManager2
  • waveform.js (optional)
  • Sushi Player (what you're here for)
